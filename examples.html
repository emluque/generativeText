fontFace example

Never mind the Bollocks!
Apply

This is an example of testing different font faces

For this to work the font faces need to be loaded in the page with and @font-face directive.

Params


	var params = {
		fontFamily: {
			values: [
			"'Alegreya Sans SC', sans-serif",
			"'Exo 2', sans-serif",
			"'Oleo Script Swash Caps', cursive",
			"'IM Fell Great Primer', serif",
			"'Gilda Display', serif",
			"'Macondo Swash Caps', cursive",
			"'PT Mono', sans-serif",
			"'Chicle', cursive",
			"'Sigmar One', cursive",
			"'Lily Script One', cursive",
			"'Denk One', sans-serif",
			"'Montaga', serif",
			"'Mystery Quest', cursive",
			"'Alike Angular', serif"
		] }
	};
