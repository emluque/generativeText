generativeText

What is it?

Generative text is a Javascript Library to apply generative design ideas on text on an HTML page.

The generativeText library allows you with some basic parameters to apply CSS properties programatically to html elements on a character by character or word by word basis.

Things you can do:

  • Rotate text character by character.

  • Bend text according to some predefined function.

  • Alternate some CSS properties on the characters of a paragraph.

  • Or you could do the same on words.

  • Randomize some properties and produce a different result on each application of the function.

    Reapply

  • And much more...

generativeText is very flexible.

    You can use generativeText on Most CSS 3 properties.
    You can combine different parameters for all kind of effects.
    You can pass predefined values, apply them randomly, on a step by step linear function or use your own math function to apply the properties on each element.
    You can add a pre-processing or post-processing function to the application of the properties.
