generativeText examples

Basic example

Randomly select a different font-family for each letter. Will produce a different result on each load of the page.

Never mind the Bollocks, here's generativeText!

Code


var params = {
	fontFamily: {
		values: [
			"'Alegreya Sans SC', sans-serif",
			"'Exo 2', sans-serif",
			"'Oleo Script Swash Caps', cursive",
			"'IM Fell Great Primer', serif",
			"'Gilda Display', serif",
			"'Macondo Swash Caps', cursive",
			"'PT Mono', sans-serif",
			"'Chicle', cursive",
			"'Sigmar One', cursive",
			"'Lily Script One', cursive",
			"'Denk One', sans-serif",
			"'Montaga', serif",
			"'Mystery Quest', cursive",
			"'Alike Angular', serif"
		]
	}
};

var ex1 = new generativeText(params);
ex1.applyToElementById("fontFace-example");

Apply to words example

Change background and foreground color, step by step and apply it to the words on the text.

Never mind the Bollocks, here's generativeText!

Code


var params2 = {
	backgroundColor: {
		type: 'list',
		values: [
			'#ca3a11',
			'#333333',
		],
		steps:true,
	},
	color: {
		type: 'list',
		values: [
			'#333333',
			'#ca3a11',
		],
		steps:true,
	}
};

var opts2 = {
	applyTo: "words",
	textSpaces: 'both',
}

var ex2 = new generativeText(params2, opts2);
ex2.applyToElementById("example2");
